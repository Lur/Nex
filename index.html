Nex

Minecraft Administration, Economy/Sign Shops, and Game Mechanics Plugin

View project onGitHub

Welcome

Nex is an Administration, Economy / Sign Shops, and Game Mechanics plugin All-In-One. Designed to be modular, you don't need to use every single one of the Nex plugins, you simply download which ones you want and place them in "/Your_Server_Directory/Plugins/Nex/"

Downloads

  • You can download the Nex Package which contains all of the Nex plugins by clicking "Download .zip file" or "Download .tar.gz file" on the right
  • PermissionsEx on bukkit
  • Latest version of Nex (Nex without any of its associated plugins)
  • SQLite and optionally SQLite Browser for viewing and editing the DB file.

How It Works

Nex acts as the core for the plugin(s), and has NexSQL built-in so it can do things like adding new entries and deleting entries from the DB. The other Nex plugins all use Nex for sending queries to the DB, handling commands, etc.

How To Use

You aren't required to use all of the Nex plugins, but they work best together. To use any of the Nex plugins you need SQLite since Nex uses an SQLite DB to store all needed data.

  1. Download whichever plugin(s) you want to use and also download the NexCore.
  2. Place your downloaded plugins and NexSQL into your /Plugins/ folder
  3. Run your server and Nex should automatically create a folder within your /Plugins/ folder, open it and edit the NexCore configuration file and the configuration file for whichever plugin(s) you downloaded.
  4. Place the "server.db" file in /Plugins/Nex/ so it looks like "/Your_Server_Directory/Plugins/Nex/server.db" Your server should now be running successfully with Nex. To view and change data like sign shops, edit the "server.db" file.

  • Nex.db.new | Nex.db.add | Nex.db.del | Nex.db.wipe

Nex Commands

*![bantime][mb|hr|dy|wk] [Player] [reason] !25mb testuser You're banned from the server for 25 minutes because you did not follow the rules.
!3dy testuser You're banned from the server for 3 days because you did not follow the rules.
!ban testuser You're permanently banned from the server because you did not follow the rules.
!xraylogs [view|clear] - View suspected x-ray users or clear out the x-ray logs from the DB.
!stafflogs [view|clear] - View suspected admin abusers or clear out the staff logs from the DB.

NexSQL Commands

Note that NexSQL is built-in to the NexCore plugin. *!db [wipe] [optional:table] - Wipe's the database or table clean. !db wipe testTbl (Would completely clear all rows from the table "testTbl")
!db wipe (Would completely clear all rows from all tables) *!db [new|add|del] [table_name] [args] !db new Players id(int),Username(string),Password(string) (Creates a new table "Players")
!db add Players 1,TestUsr,TestPwd123 (Adds a new row within the table "Players")
!db del Players Players 1 (Deletes a specific row from "Players" using only the ID specified)
