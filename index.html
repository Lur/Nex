Welcome

Nex is an Administration, Economy / Sign Shops, and Game Mechanics plugin All-In-One. Disable which features you don't want via the configuration file, e.g if you wanted to disable the NexMechanics you could change the line "NexMechanics_Enabled = true" to "NexMechanics_Enabled = false" in the file "/Plugins/Nex/config.ini"

Downloads

  • You can download the Nex Package which contains all of the Nex plugins by clicking "Download .zip file" or "Download .tar.gz file" on the right
  • PermissionsEx on bukkit

  • .sql file to create the required MYSQL DB

How It Works

Nex uses a MYSQL database to save & load player data, shop data, and anything else that might need saved.

How To Use

  1. Download the Nex Plugin, and make sure you have a MYSQL db setup.
  2. Place the plugin in /Server_Directory/Plugins/
  3. If you haven't already, upload the .SQL file via phpmyadmin to create the required mysql db.
  4. Run your server and Nex should automatically create a folder within your /Plugins/ folder, open it and edit the configuration.ini which will be where you enter the mysql db information. Your server should now be running successfully with Nex. Remember you can disable plugins/features via the configuration.ini, simply set it to disabled.

  • Nex.db.new | Nex.db.add | Nex.db.del | Nex.db.wipe

Nex Commands

  • ![bantime][mb|hr|dy|wk] [Player] [reason] !25mb testuser You're banned from the server for 25 minutes because you did not follow the rules.
    !3dy testuser You're banned from the server for 3 days because you did not follow the rules.
    !ban testuser You're permanently banned from the server because you did not follow the rules.
    !xraylogs [view|clear] - View suspected x-ray users or clear out the x-ray logs from the DB.
    !stafflogs [view|clear] - View suspected admin abusers or clear out the staff logs from the DB.

NexSQL Commands

  • !db [wipe] [optional:table] - Wipe's the database or table clean. !db wipe testTbl (Would completely clear all rows from the table "testTbl")
    !db wipe (Would completely clear all rows from all tables)
  • !db [new|add|del] [table_name] [args] !db new Players id(int),Username(string),Password(string) (Creates a new table "Players")
    !db add Players 1,TestUsr,TestPwd123 (Adds a new row within the table "Players")
    !db del Players Players 1 (Deletes a specific row from "Players" using only the ID specified)
