Welcome

Nex is an Administration, Economy / Sign Shops, and Game Mechanics plugin All-In-One. Designed to be modular, you don't need to use every single one of the Nex plugins, you simply download which ones you want and place them in "/Your_Server_Directory/Plugins/Nex/"

Downloads

  • Latest version of Nex Pack (Nex and all of its associated plugins)
  • Latest version of Nex (Nex without any of its associated plugins)
  • SQLite and SQLite Browser for viewing and editing the DB file.

How To Use

You aren't required to use all of the Nex plugins, but they work best together. To use any of the Nex plugins you need SQLite since Nex uses an SQLite DB to store all needed DB.

  1. Download whichever Nex plugins you want to use.
  2. Place them in your MC server's /Plugins/ folder
  3. Run your server and Nex should automatically create a folder within your /Plugins/ folder, open it and edit the configuration file.
  4. Place the "server.db" file in /Plugins/Nex/ so it looks like /Your_Server_Directory/Plugins/Nex/server.db Your server should now be running successfully with Nex. To view and change data like sign shops, edit the "server.db" file.
